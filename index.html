サンプルコード

　　 void setup(){

　　 　size(480, 120);

　　 }

　　 　 　

　　 void draw(){

　　 　background(176, 204, 226);

　　 　for (int x=35; x　　 　　owl(x, 110);

　　 　}

　　 }

　　 　

　　 void owl(int x, int y){

　　 　pushMatrix();

　　 　translate(x, y);

　　 　stroke(138, 138, 125);

　　 　strokeWeight(70);

　　 　line(0, -35, 0, -65);

　　 　noStroke();

　　 　fill(255);

　　 　ellipse(-17.5, -65, 35, 35);

　　 　ellipse(17.5, -65, 35, 35);

　　 　 arc(0, -65, 70, 70, 0, PI);

　　 　fill(51, 51, 30);

　　 　ellipse(-14, -65, 8, 8);

　　 　ellipse(14, -65, 8, 8);

　　 　quad(0, -58, 4, -51, 0, -44, -4, -51);

　　 　popMatrix();

　　 　}

説明文

　関数は自分で作ることができる。

　今回のプログラムではowl()という関数を作り、その関数の処理を[void owl()]で行っている。

　owl()関数が読み込まれるたびに[void　owl()]の処理を毎回行う。

　

　このプログラムではowl()関数でフクロウの図形を作成し、owl()関数を呼び出すたびにフクロウの図形を呼び出せるようになっている。

　owl(x, y)関数のx,yを指定することにより、[void　owl()]の処理にxとyの値を呼び出すことができる。

　このプログラムでは、for関数により変動するxの値とyの110という値が、int型によって使われている。
